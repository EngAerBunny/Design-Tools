This folder contains the model of the UAV used to obtain the parasite drag in OpenVSP v3.26.1

OpenVSP is a free softare from NASA and can be downloaded here: http://openvsp.org/download.php

To use and check this model:
1 - Copy the current folder into the OpenVSP folder
2 - Open the file: vsp.exe
3 - In the program go File-Open, select the downloaded folder and the CAD.vsp3 file

After this file is opened, the user can check the parasite drag calculator which adds the drag from non slender bodies which was added amnnualy based on empirical expresisons
